To build xylem with MinSizeRel for smallest binary on Linux:

1. Clean the build directory:

```bash
rm -rf out
```

2. Configure with CMake using Ninja:

```bash
cmake -B out -G Ninja -DCMAKE_BUILD_TYPE=MinSizeRel
```

3. Build:

```bash
cmake --build out -j $(nproc)
```

4. Copy compile_commands.json:

```bash
cp -f out/compile_commands.json compile_commands.json 2>/dev/null || true
```
