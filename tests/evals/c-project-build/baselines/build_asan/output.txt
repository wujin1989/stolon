To build xylem with AddressSanitizer and run tests:

1. Clean the build directory:

```bash
rm -rf out
```

2. Configure with ASAN enabled:

```bash
cmake -B out -G Ninja \
  -DCMAKE_BUILD_TYPE=Debug \
  -DXYLEM_ENABLE_TESTING=ON \
  -DXYLEM_ENABLE_ASAN=ON
```

3. Build:

```bash
cmake --build out -j $(nproc)
```

4. Run tests:

```bash
ctest --test-dir out --output-on-failure
```

Note: ASAN and TSAN cannot be enabled simultaneously -- they conflict at the compiler level.
