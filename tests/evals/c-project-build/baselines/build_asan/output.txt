To build xylem with AddressSanitizer and run tests:

1. Configure with ASAN enabled (requires Debug build):

```bash
cmake -B out -G Ninja \
  -DCMAKE_BUILD_TYPE=Debug \
  -DXYLEM_ENABLE_TESTING=ON \
  -DXYLEM_ENABLE_ASAN=ON
```

2. Build:

```bash
cmake --build out -j $(nproc)
```

3. Run tests:

```bash
ctest --test-dir out --output-on-failure
```

Note: ASAN and TSAN cannot be enabled simultaneously -- they conflict. If you need thread sanitizer, do a separate clean build with only TSAN. You can combine ASAN with UBSAN though.
