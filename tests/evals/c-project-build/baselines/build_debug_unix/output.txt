To build xylem in Debug mode on Linux:

1. Clean the build directory:

```bash
rm -rf out
```

2. Configure with CMake using Ninja:

```bash
cmake -B out -G Ninja \
  -DCMAKE_BUILD_TYPE=Debug \
  -DXYLEM_ENABLE_TESTING=ON
```

3. Build:

```bash
cmake --build out -j $(nproc)
```

4. Copy compile_commands.json to the project root for clangd:

```bash
cp -f out/compile_commands.json compile_commands.json 2>/dev/null || true
```

5. Run tests:

```bash
ctest --test-dir out --output-on-failure
```
