To generate code coverage for xylem on Linux:

1. Clean the build directory:

```bash
rm -rf out
```

2. Configure with coverage and testing enabled:

```bash
cmake -B out -G Ninja \
  -DCMAKE_BUILD_TYPE=Debug \
  -DXYLEM_ENABLE_TESTING=ON \
  -DXYLEM_ENABLE_COVERAGE=ON
```

3. Build:

```bash
cmake --build out -j $(nproc)
```

4. Generate coverage report:

```bash
cmake --build out --target coverage
```

HTML report will be at `out/coverage/html/index.html`.
