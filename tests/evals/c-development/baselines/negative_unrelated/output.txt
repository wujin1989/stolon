Here's a Python script that reads a CSV and plots a bar chart:

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("data.csv")
df.plot(kind="bar", x="category", y="value")
plt.title("Chart")
plt.tight_layout()
plt.savefig("chart.png")
plt.show()
```
