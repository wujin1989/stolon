Here's the release checklist for my-utils version 1.0.0:

1. All tests pass:
   ctest --test-dir out --output-on-failure

2. ASAN clean — no memory errors:
   cmake -B out -DMY_UTILS_ENABLE_ASAN=ON
   cmake --build out
   ctest --test-dir out --output-on-failure

3. UBSAN clean — no undefined behavior:
   cmake -B out -DMY_UTILS_ENABLE_UBSAN=ON
   cmake --build out
   ctest --test-dir out --output-on-failure

4. No compiler warnings (build with -Werror or /WX).

5. No unreplaced {project} or {PROJECT} placeholders in source files. Run:
   grep -rn '{project}\|{PROJECT}' src/ include/ tests/

6. Update the version in CMakeLists.txt:
   project(MY_UTILS VERSION 1.0.0 LANGUAGES C)

7. Follow Semantic Versioning (semver): MAJOR.MINOR.PATCH. Since this is the
   first stable release, 1.0.0 is correct.
