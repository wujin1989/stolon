To run your tests with AddressSanitizer, use the CMake option:

```bash
cmake -B out -DMYUTILS_ENABLE_ASAN=ON
cmake --build out
ctest --test-dir out --output-on-failure
```

Do not add `-fsanitize=address` flags manually. Use the project's `apply_sanitizer` helper instead.

Note: ASAN and TSAN cannot be enabled simultaneously. Run them in separate builds.
